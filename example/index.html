LipstickCSS

Color CSS Library

LipstickCSS is a CSS color library which enables to handle easily colors in HTML documents thanks to class collections.

Download

Download the compressed, production LipstickCSS Production

Download the uncompressed, development LipstickCSS Development

Usage

Background Color

In order to colorize a background, add the color name and light/dark/accent suffix (if necessary). CSS Pseudo-classes can be used thanks to their suffix (-hover, -active, -visited, -focus).

Examples

  • purple-l4 corresponds to the fourth light purple (#E1BEE7).
  • cyan-d3 corresponds to the third dark cyan (#00838F).
  • amber-a1-active corresponds to the first accent amber (#FFE57F) in the active state.

Code

‹div class="alert purple-l4 purple-l3-hover purple-d3-text"›
  May the Force be with you !
‹/div›

Result

May the Force be with you !

Text Color

In order to colorize a text, add the color name, light/dark/accent suffix (if necessary) and the -text suffix. CSS Pseudo-classes can be used thanks to their suffix (-hover, -active, -visited, -focus).

Examples

  • purple-l4-text corresponds to the fourth light purple (#E1BEE7).
  • cyan-d3-text corresponds to the third dark cyan (#00838F).
  • amber-a1-text-active corresponds to the first accent amber (#FFE57F) in the active state.

Code

‹div class="alert blue-l5 blue-text blue-d3-text-hover"›
  May the Force be with you !
‹/div›

Result

May the Force be with you !

Border Color

In order to colorize a border, add the color name, light/dark/accent suffix (if necessary) and the -border suffix. CSS Pseudo-classes can be used thanks to their suffix (-hover, -active, -visited, -focus).

border-width-nth is a class which defines the border width with nth pixels. nth can be chosen between 1 and 10.

Examples

  • purple-l4-border corresponds to the fourth light purple (#E1BEE7).
  • cyan-d3-border corresponds to the third dark cyan (#00838F).
  • amber-a1-border-active corresponds to the first accent amber (#FFE57F) in the active state.

Code

‹div class="alert green-l4 green-d3-text green-d1-border green-d3-border-hover border-width-4"›
  May the Force be with you !
‹/div›

Result

May the Force be with you !

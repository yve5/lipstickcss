LipstickCSS

Color CSS Library

LipstickCSS is a CSS color library which enables to handle easily colors in HTML documents thanks to class collections.

Usage

Background Color

In order to colorize a background, add the color name and bright/dark class (if necessary) to the element.

Code

‹div class="purple bright-4"›May the Force be with you !‹/div›

Result

May the Force be with you !

Text Color

In order to colorize a text, add the color name with the -text suffix and bright/dark class (if necessary) with the text- prefix to the element.

Code

‹div class="blue-text text-darken-1"›May the Force be with you !‹/div›

Result

May the Force be with you !
